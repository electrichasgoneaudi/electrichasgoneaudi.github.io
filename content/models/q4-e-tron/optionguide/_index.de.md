Der folgende Leitfaden basiert auf dem Basismodell: Viele Märkte haben ein Zierpaket mit Optionen.

Sie sollten das Ganze durchlesen [option list](list) Aber unten sind die wichtigsten Optionen, die Sie beim Kauf eines Audi Q4 e-tron beachten müssen.

![IU1](https://media.evkx.net/ehga/models/q4-e-tron/optionguide/iu1.webp)

## Smartphone-Schnittstelle: Option ID IU1

Ohne Smartphone-Schnittstelle können Sie Android Auto / Apple Carplay nicht verwenden und dann Apps wie Waze oder Spotify gut steuern.

Aktuell ist dies nur im Infotainmentpaket mit optionID verfügbar **PYX** Dazu gehören auch die Telefonbox und zwei USB-C auf Rücksitzen.

[See smartphone interface for details.](../technology/uiandoperations/smartphoneinterface/)

![QL2](https://media.evkx.net/ehga/models/q4-e-tron/optionguide/ql5.webp)

## Datenschutzfenster: Option ID QL5

Verbesserungen sieht viel aus und Sie müssen nicht in einem ausgestellten Fischtank herumfahren.

[See windows for details](../exterior/windows/#privacy-glass)

![PYG](https://media.evkx.net/ehga/models/q4-e-tron/optionguide/pyg.webp)

## MMI Navigation plus: Option ID PYG

Mit der Option MMI Navigation Plus erhalten Sie MMI Navigationssystem und virtuelles Cockpit. Ein eingebautes Navigationssystem bietet mehrere Vorteile gegenüber der Verwendung Ihres Telefons.

Siehe Einzelheiten zu [MMI Navigation Plus](../technology/uiandoperations/navigation/) und [Virtual Cockpit](../technology/uiandoperations/virtualcockpit/)

![Matrix](https://media.evkx.net/ehga/models/q4-e-tron/technology/lights/signature1s.webp)

## Matrix-LED-Scheinwerfer

Mit Matrix LED-Scheinwerfern erhalten Sie auf Landstraßen deutlich verbesserte Lichter. Darüber hinaus ist Matrix LED-Scheinwerfer eine Voraussetzung, um den kühlen roten Streifen in den Hintergrund zu bringen.

Lesen Sie mehr über [Matrix LED Headlights](../technology/lights/#matrix-headlights).

## Assistantpackage Pro: Option ID PY7

Das Assistenzpaket Pro enthält die meisten wichtigen Fahrerassistenzfunktionen wie [Adaptive Cruise Assist](../technology/drivingassistance/adaptivecruiseassist/), [Adaptive Cruise Control](../technology/drivingassistance/adaptivecruisecontrol/)  und [Parking System Plus](../technology/drivingassistance/parkingsystemplus/)

## Sicherheitspaket plus: Option ID PYU

Der Seitenassistent ist nicht Teil des Assistantpackage Pro, sondern im Sicherheitspaket plus Paket enthalten. [Side Assist](../technology/drivingassistance/sideassist/), [exit warning](../technology/drivingassistance/exitwarning/), [cross traffic assist rear](../technology/drivingassistance/crosstrafficassistrear/), [Audi Pre Sense rear](../technology/drivingassistance/presenserear/).

![QQ2](https://media.evkx.net/ehga/models/q4-e-tron/optionguide/qq2.webp)

## Mehrfarbige Umgebungsbeleuchtung: Option ID QQ2

Mit mehrfarbiger Umgebungsbeleuchtung erreicht das Aussehen und Gefühl im Auto einen anderen Planeten.

[See ambient lighting to get convinced](../interior/lights/)
