Das fahrzeugseitige Ladegerät ist für die Umwandlung von Wechselstrom in Gleichstrom verantwortlich.

Die standardmäßige Onboard-Ladegerät-Unterstützung bis zu 11 kW AC-Ladung

In den USA hat der Ladehafen einen [J1772 connector](https://en.wikipedia.org/wiki/SAE_J1772) um mit dem Auto zu verbinden, während es in Europa eine [Type 2 connector](https://en.wikipedia.org/wiki/Type_2_connector).

### Optionales 22KW Ladegerät

Sie können ein zusätzliches Ladegerät für das Auto bestellen (AX5). Dies gibt dem Auto 22KW Ladefähigkeit auf 400Volt 32A 3 Phasen Verbindung.

Sie müssen den zusätzlichen Port bestellen **JS1** und die [Audi Charging System Connect](/models/e-tron/technology/chargingsystem/#e-tron-charging-system-connect) **NW2**

### Kapazität basierend auf Netzwerk / Outlet

| Anschluss | Stecker  | Kapazität | Aufladen 100% e-tron 55 |
| ------| ------| ---- |------- |
| 120Volt | Ebene 1 Inländische NEMA 5-20 | 1,2 kW |  76 Stunden |
| 240Volt | Inländische NEMA 14-50 | 9,6 kW |  9,5 Stunden |
| 230Volt | Inländischer Typ F | 1,8kW |  50,5 Stunden |
| 400V 32A 3phasig | Rotindustriell |  22KW | 4,5 Stunden |
| 400V 16A 3phasig | Rotindustriell |  11KW | 9 Stunden |
| 230V 32A 1 Phase | Blauindustriell |  7,2 KW | 11,5 Stunden |
| 230V 16A 1 Phase | Blauindustriell |  3,6 KW | 23 Stunden |
