The Audi Q5 e-tron is offered in two variants, both using the same 83.4 kWh battery but differing in power output and drivetrain.

{{< evkxfiguresized thumb="models/audi/q5_e-tron/q5_e-tron_50_quattro/exterior_b7b55_st.webp" width="1920" height="1080" title="Audi Q5 e-tron" >}}

## Audi Q5 e-tron 40

The entry variant with rear-wheel drive and a single electric motor.

| Specification | Value |
|---|---|
| Peak power | 150 kW (204 hp) |
| Torque | 310 Nm |
| 0–100 km/h | 9.3 s |
| Top speed | 160 km/h |
| CLTC range | 560 km |
| CLTC consumption | 15.9 kWh/100 km |
| Battery (gross) | 83.4 kWh |
| Battery (net) | 76.6 kWh |
| Battery chemistry | NMC |
| Drive | Rear-wheel drive |

## Audi Q5 e-tron 50 quattro

The performance variant with dual electric motors and quattro all-wheel drive.

| Specification | Value |
|---|---|
| Peak power | 225 kW (306 hp) |
| Torque | 460 Nm |
| 0–100 km/h | 6.7 s |
| Top speed | 160 km/h |
| CLTC range | 550 km |
| CLTC consumption | 16.2 kWh/100 km |
| Battery (gross) | 83.4 kWh |
| Battery (net) | 76.6 kWh |
| Battery chemistry | NMC |
| Drive | All-wheel drive (quattro) |

## Shared specifications

Both variants share the following:

| Specification | Value |
|---|---|
| Length | 4,876 mm |
| Width (excl. mirrors) | 1,860 mm |
| Height | 1,675 mm |
| Wheelbase | 2,965 mm |
| Ground clearance | 180 mm |
| Seating | Six or seven seats across three rows |
| Maximum cargo volume | Up to 1,825 L |
| Max roof load | 75 kg |
| Max trailer weight (braked) | 1,200 kg |
| Charge port location | Left rear side |
