Audi Q5 e-tron tilbys i to varianter, begge med samme 83,4 kWh batteri, men med forskjellig effekt og drivlinje.

## Audi Q5 e-tron 40

Grunnvarianten med bakhjulsdrift og én elektrisk motor.

| Spesifikasjon | Verdi |
|---|---|
| Toppeffekt | 150 kW (204 hk) |
| Dreiemoment | 310 Nm |
| 0–100 km/t | 9,3 s |
| Toppfart | 160 km/t |
| CLTC rekkevidde | 560 km |
| CLTC-forbruk | 15,9 kWh/100 km |
| Batteri (brutto) | 83,4 kWh |
| Batteri (netto) | 76,6 kWh |
| Batterikjemi | NMC |
| Drift | Bakhjulsdrift |

## Audi Q5 e-tron 50 quattro
