Der Audi Q5 e-tron wird in zwei Varianten angeboten, die beide den gleichen 83,4 kWh Akku verwenden, sich aber in Leistung und Antriebsstrang unterscheiden.

{{< evkxfiguresized thumb="models/audi/q5_e-tron/q5_e-tron_50_quattro/exterior_b7b55_st.webp" width="1920" height="1080" title="Audi Q5 e-tron" >}}

## Audi Q5 e-tron 40

Die Einstiegsvariante mit Hinterradantrieb und einem einzigen Elektromotor.

| Spezifikation | Wert |
|---|---|
| Spitzenleistung | 150 kW (204 PS) |
| Drehmoment | 310 Nm |
| 0–100 km/h | 9,3 s |
| Höchstgeschwindigkeit | 160 km/h |
| CLTC-Bereich | 560 km |
| CLTC-Verbrauch | 15,9 kWh/100 km |
| Batterie (brutto) | 83,4 kWh |
| Batterie (netto) | 76,6 kWh |
| Batteriechemie | NMC |
| Antrieb | Hinterradantrieb |

## Audi Q5 e-tron 50 quattro

Die Performance-Variante mit Doppel-Elektromotoren und quattro Allradantrieb.

| Spezifikation | Wert |
|---|---|
| Spitzenleistung | 225 kW (306 PS) |
| Drehmoment | 460 Nm |
| 0–100 km/h | 6,7 s |
| Höchstgeschwindigkeit | 160 km/h |
| CLTC-Bereich | 550 km |
| CLTC-Verbrauch | 16,2 kWh/100 km |
| Batterie (brutto) | 83,4 kWh |
| Batterie (netto) | 76,6 kWh |
| Batteriechemie | NMC |
| Antrieb | Allradantrieb (quattro) |

## Gemeinsame Spezifikationen

Beide Varianten haben folgende Gemeinsamkeiten:

| Spezifikation | Wert |
|---|---|
| Länge | 4,876 mm |
| Breite (ausg. Spiegel) | 1,860 mm |
| Höhe | 1,675 mm |
| Radstand | 2965 mm |
| Bodenfreiheit | 180 mm |
| Sitzplätze | Sechs oder sieben Sitze in drei Reihen |
| Maximales Frachtvolumen | Bis zu 1.825 L |
| Maximale Dachlast | 75 kg |
| Maximales Gewicht des Anhängers (gebremst) | 1.200 kg |
| Ladeanschluss-Standort | linke Rückseite |
