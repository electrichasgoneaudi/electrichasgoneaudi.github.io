## Automatische Verriegelung

Sie können Türen beim Fahren verriegeln. Diese Einstellungen finden Sie im MMI

## Kindersicherheit

Vom Fahrersitz aus können Sie Kindersicherheit für Heckfenster und Türen ermöglichen.

## Heckklappe mit Antrieb

Sie können das Auto mit einer angetriebenen Heckklappe bestellen.

## Einfahrbeleuchtung

Sie können den e-tron GT-Türen coole Eingangsleuchten hinzufügen.

## Keyless-Einstieg

Es ist Standard mit [advance key](../../technology/lockingsystems/#advance-key-option-pgc) Das gibt Ihnen schlüssellosen Zugang zum Auto

## Heckklappe des Kicksensors

Wenn Sie die [advance key option](../../technology/lockingsystems/#advance-key-option-pgc) Sie erhalten Kick-Sensor für die Heckklappe

## Verriegelung von der Heckklappe

Wenn Sie die [advance key option](../../technology/lockingsystems/#advance-key-option-pgc) Sie erhalten einen Schließknopf an der Heckklappe, damit Sie das Auto verriegeln können.
