Some users have experienced installed apps disappearing from one MMI profile while remaining available in other profiles. If an app such as Spotify or Vivaldi is missing from the main menu, it may still be listed as installed in the privacy settings.

Until the underlying software issue is resolved, uninstalling and reinstalling the hidden app can restore it:

1. Open **Privacy settings**.
2. Select **Service overview**, then **Installed infotainment services**.
3. Select **Show all apps**.
4. Find the missing app and choose **Uninstall**.
5. Confirm with **OK**.
6. Return to the app store and install the app again.
