Audi e-tron drive select

Audi e-tron ist serienmäßig mit drive select ausgestattet.

Zuletzt aktualisiert 15.07.2026
Audi e-tron Varianten Abmessungen Außen Spezifikationen Innenausstattung Antriebsstrang Tranportationsfähigkeiten Technologie Modelljahresänderungen Multimedia Überprüfungen Options Guide Wissensaustausch

Das dynamische Fahrverhalten von Audi drive select ermöglicht es dem Fahrer, verschiedene Einstellungen in seinem Audi e-tron zu erleben. Die Wahl eines der Modi Auto, Komfort, Allroad, Offroad, Dynamik oder Effizienz verändert die Charakteristik wichtiger Antriebs- und Fahrwerkskomponenten von einer Komfortvorspannung zu einer deutlich dynamischen oder besonders energieeffizienten.

Im individuellen Modus können Fahrer das Setup nach ihren persönlichen Vorlieben konfigurieren.

Abhängig von Option und Setup beeinflussen die Einstellungen der Laufwerksauswahl die folgenden:

  • Motorreaktionen
  • quattro-Antrieb
  • Adaptive Luftfederung mit elektronischer Dämpfersteuerung
  • dynamisches Lenkrad
  • HD Matrix LED-Scheinwerfer
  • Umgebungslicht
  • Elektronische Stabilisierungskontrolle (ESC)
  • Adaptiver Cruise Assist
  • Audi Pre Sense

Modi

Offroad - Bietet Unterstützung beim Fahren im Gelände. Das Fahrzeug wird mit der Luftfederung® auf die zweithöchste Ebene angehoben. Der ESC wird eingeschränkt und der Bergabstieg unterstützt* Das Fahrzeug wird nach dem Ausschalten der Zündung aus diesem Zustand ausgeschaltet.

All-Wege - Bietet ein ausgewogenes Fahrverhalten. Das Fahrzeug wird mit der Luftfederung auf die zweithöchste Ebene angehoben*.

Effizienz - Das Fahrzeug schaltet auf eine kraftstoffeffiziente Fahrzeugeinstellung um und unterstützt den Fahrer mit Kraftstoffverbrauchsmeldungen. Das Fahrprogramm „E ist ausgewählt.

Komfort - Bietet eine komfortorientierte Fahrzeugeinstellung und eignet sich für lange Fahrten auf Autobahnen.

Auto - Bietet ein insgesamt komfortables und dennoch dynamisches Fahrgefühl und eignet sich für den täglichen Gebrauch.

dynamisch - Verleiht dem Fahrer ein sportliches Fahrgefühl und ist für einen sportlichen Fahrstil geeignet. Das Fahrprogramm „S ist ausgewählt.

Einzelperson - Bietet die Option für personalisierte Fahrzeugeinstellungen.

Einzelne Einstellungen

Antrieb

Das Antriebssystem reagiert schneller oder ausgeglichener auf Fahrpedalbewegungen.

Lenkung

Die Servolenkung passt sich an. Leichte Lenkung eignet sich beispielsweise für lange Fahrten auf Autobahnen oder für das reibungslose Fahren um Kurven.

Suspension

Die Federung stellt sich beim Ausgleich von Unebenheiten auf der Straße fester oder komfortabler ein, die Bodenfreiheit des Fahrzeugs wird durch die Luftfederung eingestellt.* basierend auf der Einstellung.
