Audi e-tron adaptiver Cruise Assist

Audi e-tron kann mit dem Adaptive Cruise Assist ausgestattet werden. Der adaptive Cruise Assist ist eine Kombination aus adaptivem Tempomat und aktivem Spurassistent.

Der adaptive Reiseassistent unterstützt den Fahrer beim Beschleunigen, Bremsen, Halten von Geschwindigkeit und Abstand sowie bei der Spurführung.

Dies kann den Fahrkomfort insbesondere auf langen Fahrten erhöhen. Das System erkennt Fahrspurmarkierungen, Seitenstrukturen, Fahrzeuge in benachbarten Fahrspuren und mehrere vorausfahrende Fahrzeuge im Rahmen von Systemgrenzen. Aus diesen Informationen leitet das System einen virtuellen Fahrweg ab und erledigt dies Fahrzeug in diesem.

Mit einem unterstützenden Lenkeingriff hilft das System, das Fahrzeug dauerhaft in der Spurmitte zu halten, gleichzeitig wird die eingestellte Geschwindigkeit konstant gehalten und das System reagiert auf vorausfahrende Fahrzeuge mit einer Abstandsregelung.

Dank der Vielzahl der beteiligten Parameter bietet der adaptive Fahrassistent das komfortabelste Fahren im Geschwindigkeitsbereich von 0 bis maximal 250 km / h und im Stau. Der adaptive Fahrerassistent nutzt den im Frontbereich des Fahrzeugs installierten Radarsensor, die Frontkamera und die Ultraschallsensoren, um den Bereich um das Fahrzeug herum permanent zu überwachen. Das System wird auch durch ein Lenkrad mit Handgrifferkennung in einem spezifischen Lenkraddesign unterstützt.
