Audi Q8 e-tron Tür & Luke

Audi hat einige Funktionen und Optionen für Türen und Luke.

Automatische Verriegelung

Sie können Türen beim Fahren verriegeln. Diese Einstellungen finden Sie im MMI

Kindersicherheit

Vom Fahrersitz aus können Sie Kindersicherheit für Heckfenster und Türen ermöglichen.

Heckklappe mit Antrieb

Alle e-tron haben eine angetriebene Heckklappe.

Türleuchten

Wenn Sie die Umgebungslichtoptionen bestellen QQ1 oder QQ2 Alle Türen sind mit Lichtern ausgestattet

Einfahrbeleuchtung

Sie können den e-tron-türen coole eintrittsleuchten hinzufügen.

OptionId ****

Servo-Schließung

Mit Servo-Schließen der Türen werden die Türen automatisch die letzten cm geschlossen.

OptionID GZ2

Keyless-Einstieg

Wenn Sie die advance key option Sie erhalten schlüssellosen Zugang zum Auto

Heckklappe des Kicksensors

Wenn Sie die advance key option Sie erhalten Kick-Sensor für die Heckklappe

Verriegelung von der Heckklappe

Wenn Sie die advance key option Sie erhalten einen Schließknopf an der Heckklappe, damit Sie das Auto verriegeln können.

Türeinlegeteile

Sie können e-tron mit verschiedenen Inlays in den Türschwellern erhalten
