Audi Q4 e-tron Tür & Luke

Audi hat einige Funktionen und Optionen für Türen und Luke.

Zuletzt aktualisiert 15.07.2026

Automatische Verriegelung

Sie können Türen beim Fahren verriegeln. Diese Einstellungen finden Sie im MMI

Kindersicherheit

Vom Fahrersitz aus können Sie Kindersicherheit für Heckfenster und Türen ermöglichen.

Heckklappe mit Antrieb

Als Option (4E6) können Sie eine angetriebene Heckklappe zum Audi Q4 e-tron erhalten. Dies ist in der Sportback-Version standardmäßig.

Keyless-Einstieg

Wenn Sie die advance key option Sie erhalten schlüssellosen Zugang zum Auto

Heckklappe des Kicksensors

Wenn Sie die advance key option zusammen mit der angetriebenen Heckklappe erhalten Sie Kick-Sensor für die Heckklappe

Verriegelung von der Heckklappe

Wenn Sie die advance key option Sie erhalten einen Schließknopf an der Heckklappe, damit Sie das Auto verriegeln können.
